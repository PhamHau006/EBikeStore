## Installation steps:

1. Install Development Tools

```bash
# Install Vue CLI
npm install -g @vue/cli

# Install Ionic CLI
npm install -g @ionic/cli
```

2. Clone the Repository

```bash
git clone https://github.com/PhamHau006/EBikeStore
cd EBikeStore
```

3. Install Dependencies

```bash
npm install
# or
yarn install
```

4. Start Development Server

```bash
ionic serve
```
